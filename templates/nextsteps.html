Javascript

About

Javascript is the programming language of the web which adds interactivity to web pages. It is one of the three languages that all web developers must learn. Javascript is used to program the behaviour of web pages where as HTML adds content and css specifies the layout. For example buttons' response to clicks, animations and games are created through javascript. On this particular page javascript and jQuery has been used for the navigation buttons.(About me, Udacity Notes, Next Steps). Javascript runs on the web browser allowing to control web page behaviour. if statements, for and while loops function the same or nearly so just like in python.

Resources

  • Codeacademy

    Codeacademy has two courses each on Javascript and jQuery. The first provides a basic introduction to javascript with explanations of concepts and has a programming environment to write and run Javascript. jQuery course that provides more visual feedback of the different functions compliments the javascript course well.

    The JQuery course seemed to be more useful for making the web application more interactive. The course showed how to show/hide elements when clicking on buttons. Or how to change the background interactively when hovering the mouse over something.

  • w3schools

    This was very useful and comprehensive tutorial when starting out with javascript. It has examples in javascript for most things that we did when learning python. For example.

    • Variables
    • Strings
    • Functions
    • Arrays, very similar to Python lists
    • For Loops
    • While Loops
    • If/else

    Furthermore, this is also very useful as a reference. When trying to remember how to use a language feature.
